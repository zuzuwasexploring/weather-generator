Icon Implementation Guide

Icon 9 - Partly Cloudy (Sun + Cloud)

<svg viewBox="0 0 100 100" xmlns="http://www.w3.org/2000/svg">
  /* Yellow sun partial */
  <circle cx="35" cy="35" r="22" fill="#FFB84D"/>
  /* White cloud covering it */
  <path d="M45 50..." fill="white"/>
</svg>

Icon 10 - Windy Cloud (Layered clouds)

/* Small cloud (top, blue tint) */
<path d="M15 45..." fill="#B3D9F2"/>
/* Large cloud (bottom, white) */
<path d="M30 60..." fill="white"/>

Icon 11 - Heavy Rain (Multiple drops)

/* Split cloud - yellow + dark */
<ellipse cx="35" cy="35" fill="#FFB84D"/>
<path d="M25 55..." fill="#4B5563"/>
/* Multiple blue raindrops */
<circle cx="30" cy="75" r="3" fill="#60A5FA"/>

Icon 12 - Rainy & Windy (Tilted drops + wind circles)

/* Dark cloud */
<path d="M25 55..." fill="#4B5563"/>
/* Tilted raindrops (transform rotate) */
<ellipse cx="28" cy="75" transform="rotate(-25)"/>
/* Wind effect circles on left */
<circle cx="15" cy="45" r="6" opacity="0.6"/>
